This is the source code for the paper 

Judith Gaspers, Philipp Cimiano. A computational model for the item-based induction of construction networks. Cognitive Science. 

As an example of how to use this program see ./src/testing/RoboCup.java which performs 4-fold cross-validation on the RoboCup soccor corpus. To run the program create the following folders

./data/robocup/train
./data/robocup/gold_standard
./data/grammars

and save the RoboCup XML files to their corresponding folders. The RoboCup corpus is available online at: https://www.cs.utexas.edu/users/ml/clamp/sportscasting/data/
